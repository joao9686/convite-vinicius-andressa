# Convite de Casamento - HTML, CSS e JS puro

Estrutura:

convite-casamento/
├─ index.html
├─ styles.css
├─ script.js
└─ assets/
   ├─ folhas-topo.png
   ├─ folhas-baixo.png
   ├─ raios-luz.png
   ├─ flores-topo.png
   ├─ flores-baixo.png
   └─ iniciais-av.png

Como usar:
1. Extraia o ZIP.
2. Abra a pasta.
3. Coloque suas imagens PNG dentro de assets com os nomes acima.
4. Dê dois cliques em index.html.

Se ainda não tiver os PNGs, o site abre mesmo assim, só sem as decorações.
